# Personal website

This is the code for my personal website [julianassmann.de](https://julianassmann.de)

## Get started

Run 

`npm install`

to install all dependencies of the project. Run

`npm run dev`

to compile and serve the project locally under http://localhost:3000.

## Made with

- [NextJS](https://nextjs.org/)
- [npm](https://www.npmjs.com/)
- [Material-UI](https://material-ui.com/)
